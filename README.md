Display-App

Automated mouse moves for keeping a managed macOS computer awake. 

1. Copy Display.app into /Applications/Utilities
2. Copy DisplayLaunch.app into /Applications
3. Open DisplayLaunch.app and leave open to launch automated workflow.
4. Quit DisplayLaunch.app to end workflow.

Modify number of seconds from default 180 to any desired number of seconds in launch.scpt to change how frequently the mouse moves and clicks.
